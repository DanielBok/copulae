Getting Started
===============

Installing Copulae
------------------

Copulae can be installed via pip from `PyPI <https://pypi.org/project/pandas>`_

.. code-block:: bash

    pip install copulae

Otherwise, install :code:`copulae` from conda forge via

.. code-block:: bash

    conda install -c conda-forge copulae
